# Tiny labelled ingestion corpus

Fixture provenance: this corpus is a curated adaptation of the preserved automatic-ingestion design bundle's inert sample-corpus ideas. It is copied and rewritten here as test data; tests never import or resolve the preserved source bundle at runtime.

The labels are hand-authored evaluation truth, not generated output. Source bytes and paths are immutable inputs whose SHA-256 digests are recorded in `expected-graph.json`.
