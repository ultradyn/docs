# Acceptance Gates

## Vertical-slice gate

- source snapshot and references verify;
- Researcher returns no answer prose;
- Evidence Critic independently accepts/rejects by facet;
- Claim Extractor cannot self-accept;
- accepted claims compose an answer with exact traceability;
- source/claim mutation invalidates the answer;
- end-to-end replay passes with provider cache/session disabled.

## Exploration gate

- every automatic child owns a novel obligation;
- forward frontier, reverse accounting, and human injections have honest terminal states;
- budget/tool failures cannot create closure;
- coverage ledger has no silent selected-unit drops.

## Publication gate

- stable claims and graph revision;
- approved information architecture and canonical homes;
- material sentences supported;
- representative navigation tasks pass;
- authority/extraction/security/rights gates pass;
- Git PR and post-merge replay succeed.

## Production gate

- labeled real-project quality and confidence thresholds;
- zero-cache cost within approved envelope;
- disaster recovery and deletion drill;
- dashboard comprehension and operator runbook;
- supported-format and data-policy scope published.
