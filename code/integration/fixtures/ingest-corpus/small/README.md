# Small labelled ingestion corpus

Fixture provenance: this corpus is a curated, shortened, stable subset adapted from this repository's `CONTEXT.md`, ADRs, engineering guidance, agent guidance, and automatic-ingestion specification. It is deliberately copied into fixture-owned source files: tests do not read live repository documentation or the preserved design bundle at runtime.

The labels are hand-authored evaluation truth. Paths and source bytes are fixed by literal SHA-256 digests in `expected-graph.json`.
