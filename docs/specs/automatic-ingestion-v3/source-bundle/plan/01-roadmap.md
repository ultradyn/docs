## Recommended release increments

- **R0 — Lab:** M0.
- **R1 — Evidence core:** M1–M3; internal CLI/API, no rich UI.
- **R2 — Exploration:** M4–M5; operator dashboard beta and provisional graph exports.
- **R3 — Publication:** M6–M7; reviewed Docent import.
- **R4 — Production:** M8; approved corpora and operating policies.
