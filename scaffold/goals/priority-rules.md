# Priority rules

Priority is a rule-selected tier, never a weighted score. Every assignment includes a visible rationale and may be directly overridden.

| Tier | Decisive rule                                                                        |
| ---- | ------------------------------------------------------------------------------------ |
| P1   | Unresolved contradiction; explicit asker rejection/reopen                            |
| P2   | New demand promotes a deferred question; active question retains an unsatisfied goal |
| P3   | Raw question default                                                                 |
| P4   | Noncontradictory generated question at depth 1                                       |
| P5   | Generated depth 2 or deeper; optional extra detail                                   |

An explicit human override takes precedence and records its rationale in provenance.
