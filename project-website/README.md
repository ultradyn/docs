# Project website placeholder

This directory contains a placeholder static site for GitHub Pages. It is only here so the repository can publish a temporary page that says "Coming soon".

When a proper project website is built, it should use whatever frameworks, tooling, and structure are appropriate for that site. The current GitHub Pages placeholder site should not be taken into consideration when making those choices.

At that time, edit the GitHub Pages workflow in `.github/workflows/pages.yml` so it builds and publishes the real website appropriately.

Also update or remove this `project-website/README.md` file when the proper project website replaces the placeholder.
